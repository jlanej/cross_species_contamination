# Golden Test Fixtures

This directory contains versioned golden-output files used for regression
testing.  Any change to these files indicates a change in pipeline behaviour
and should be reviewed carefully.

## Files

| File | Module | Description |
|------|--------|-------------|
| `aggregate_matrix.tsv` | aggregate | Expected raw-count matrix for the two-sample fixture |
| `aggregate_metadata.json` | aggregate | Expected aggregation metadata |
| `detect_flagged_samples.tsv` | detect | Expected flagged-sample table (MAD method) |
| `detect_qc_summary.json` | detect | Expected QC summary JSON |
| `detect_quarantine_list.txt` | detect | Expected quarantine list |

## Regenerating

Golden files are regenerated by running:

```bash
python tests/generate_golden.py
```

Always inspect the diff before committing updated golden files.

## Input Data

Golden outputs are produced from deterministic synthetic inputs defined in
`tests/test_golden.py`.  The aggregate golden uses two Kraken2 reports
(sampleA and sampleB) with known read counts.  The detect golden uses a
six-sample matrix with one spiked contaminated sample (`contam_01`).
